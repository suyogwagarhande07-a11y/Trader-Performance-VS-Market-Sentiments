# Trader Performance vs Market Sentiment Analysis

## 📌 Project Overview
This project investigates the correlation between market sentiment regimes (Fear & Greed Index) and individual trader profitability. It includes a full data pipeline from preprocessing to predictive modeling.

---

## 🔗 Dataset Access
The primary dataset is **45MB**, which exceeds GitHub's file size limits. 
* **Download the CSV here:** [https://drive.google.com/file/d/10c_Motnr8_KYG9pgC5Je3ij7PEeiHbRj/view?usp=sharing]

---

## 💡 Key Data Insights
* **Win Rate Alpha:** "Extreme Greed" is the most profitable regime, yielding a **54.7% win rate**.
* **Sentiment Bias:** Top-tier traders exhibit a **53.3% long bias** during "Extreme Fear" periods.
* **Predictive Power:** Developed a Random Forest model achieving **72.5% accuracy** in forecasting trader profitability.

## 🚀 How to Run
1. Clone this repository to your local machine.
2. Download the `trader_analysis_with_filters.csv` from the Google Drive link above.
3. Place the CSV in the root directory.
4. Open `primetrade_assignment.ipynb` in VS Code or Jupyter Notebook.
